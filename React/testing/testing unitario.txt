Que es un testing unitario?

Un testing unitario se encarga de comprobar el funcionamiento de una pieza de codigo especifica de manera aislada. No testeamos toda la aplicacion, sino un componente especifico o solo una funcion

Para hacer testing en react, se utiliza una libreria llamada "jest" que ya viene instalada por defector dentro de las dependencias que instala "create react-app"

----------------------------------------------------------
